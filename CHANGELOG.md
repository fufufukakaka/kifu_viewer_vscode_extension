# Changelog

本拡張機能の変更履歴。フォーマットは [Keep a Changelog](https://keepachangelog.com/ja/1.1.0/) に準拠し、[Semantic Versioning](https://semver.org/lang/ja/) を採用しています。

## [0.0.1] - 2026-04-20

### Added
- `.kif` ファイルを盤面として表示するコマンド `Kifu: Show Preview` / `Kifu: Show Preview to the Side`
- 手順のナビゲーション（⏮ ← → ⏭ ボタン・スライダー・キーボード）
- 移動元／移動先セルのハイライト
- 手一覧のクリックジャンプ、ダブルクリックで原文の該当行にフォーカス
- 局面ごとのコメント表示・編集・書き戻し（`*` 行を WorkspaceEdit で更新）
- 平手／香落ち／左右香落ち／角落ち／飛車落ち／飛香落ち／二枚〜十枚落ちの初期盤面生成
- 駒落ち時の上手先手（手番パリティ反転）に対応
- `先手`/`後手`／`下手`/`上手` のプレイヤー表記切替
- ファイル保存・編集時のライブリロード
